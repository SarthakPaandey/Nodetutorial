# Node.js Tutorial

A beginner-friendly Node.js learning project covering core Node.js concepts and file system operations.

## Overview

This repository is a hands-on tutorial project for learning Node.js fundamentals. It covers basic Node.js operations including working with the file system, serving HTML files, and setting up a simple HTTP server.

## Topics Covered

- Node.js file system module (`fs`)
- Serving static HTML files
- Basic HTTP server setup
- Directory operations

## Files

- `index.js` — Main Node.js script
- `index.html` — Sample HTML file
- `dir/` — Directory for file system practice

## Getting Started

```bash
node index.js
```
